Netests.io is available via `pip`

## Installation

```shell
pip install netests
```



## Virtual Environment

It is possible (not an obligation) to use a python virtual environnement.

```shell
mkdir netests-demo
cd netests-demo

python3 -m venv .
pip install --upgrade pip

pip install netests
```
